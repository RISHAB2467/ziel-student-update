# Quick Start - Search & Download Features

## 🚀 Getting Started (2 Minutes)

### Step 1: Open Teacher Page
- Navigate to the Teacher Panel
- You'll see a new "Search and Filter" section above your entries

### Step 2: Try the Search

#### Option A: Search by Name
```
1. Click "Search by Teacher Name" field
2. Type the first letter of a name
3. Watch autocomplete suggestions appear
4. Click a suggestion or keep typing
5. Results filter instantly!
```

#### Option B: Search by Date
```
1. Click "Search by Date" field
2. Pick a date from the calendar
3. Results show only that date's entries
```

#### Option C: Search by Both
```
1. Enter a teacher name
2. Select a date
3. See only entries matching both!
```

### Step 3: Download Your Data

#### Quick PDF Download:
```
1. Make sure some checkboxes are selected (default: all checked)
2. Click "📄 Download as PDF"
3. Done! PDF downloads automatically
```

#### Quick Excel Download:
```
1. Select fields you want (or use default)
2. Click "📊 Download as Excel"
3. Done! Excel file downloads
```

#### Weekly Data Download:
```
1. Select fields to include
2. Click "📅 Download Weekly Data"
3. Get organized Excel with week-wise sheets!
```

## 💡 Pro Tips

### Tip 1: Clear Everything
Can't find what you're looking for?
```
Click "Clear Filters" button → Everything resets
```

### Tip 2: Custom Downloads
Only want specific fields?
```
Uncheck fields you don't need → Download → Smaller, focused file
```

### Tip 3: Filter Before Download
Want a specific subset?
```
Apply filters → Download → Only filtered data exports
```

### Tip 4: Weekly Reports
End of week routine:
```
1. Don't apply any filters (to get all data)
2. Select all relevant fields
3. Click "Download Weekly Data"
4. Share with admin/team
```

## ⚠️ Quick Troubleshooting

→ Clear filters and try again

→ Make sure at least one field is checked

### Autocomplete doesn't show
→ Type at least one character

### Nothing downloads
→ Check if you have data in the last 15 days

## 📊 What Gets Downloaded?

### Filtered Downloads (PDF & Excel):
- Only the entries currently visible on screen
- If you searched for "John" → Only John's data exports
- If you picked a date → Only that date exports
- If both → Only entries matching both

### Weekly Download:
- ALL data from last 15 days (ignores filters)
- Organized by week automatically
- Each week in a separate Excel sheet

## ❓ Quick FAQ

**Q: Can I search multiple teachers at once?**
A: Not yet - select one teacher at a time

**Q: Can I select a date range?**
A: Currently single date only (feature planned)

**Q: What if I have more than 15 days of data?**
A: System shows last 15 days - older entries auto-hide

**Q: Can I download without selecting any fields?**
A: No - at least one field must be checked

**Q: Do filters apply to weekly download?**
A: No - weekly download includes ALL data organized by week

**Q: Can I save my filter settings?**
A: Not yet (future enhancement)
